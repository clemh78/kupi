F.A.Q

Comment les points sont-ils distribués ?

Pour chaque pronostic, 3 cas de figures sont possibles :

  • Vous avez mis le score exact (ex : 2-0 pronostiqué, 2-0 en réalité), alors dans ce cas vous recevrez 40 points
  • Vous avez un écart identique (ex : 2-0 pronostiqué, 3-1 en réalité), alors dans ce cas vous recevrez 20 points
  • Vous avez le bon vainqueur (ex : 2-0 pronostiqué, 1-0 en réalité), alors dans ce cas vous recevrez 10 points
Pour les matchs à partir des 8ème de finales (match nul non autorisé), le score que vous renseignez correspond au score avant la séance de T.A.B (si pas de prolongation, alors correspond au score à la 90').

S'ajoutent à cela, les paris bonus :
  • Pour une équipe en finale et à la bonne place : 100 points
  • Pour une équipe en finale mais pas à la bonne place : 50 points

Les scores seront-ils rafraîchis en direct ?

Oui les scores seront rafraîchis avec un interval d'une minute.

Est-il possible d'inviter mes amis sur Kupi.fr ?

Oui vous avez la possibilité de créer un salon puis de partager le lien d'inscription à ce salon.

J'ai trouvé un bug ou j'ai une idée d'amélioration, est-il possible de contacter l'équipe ?

Oui vous avez la possibilité d'ouvrir une issue sur le projet GitHub de Kupi. L'adresse du GitHub est la suivante : https://github.com/clemh78/worldcup
